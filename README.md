# PCgene

PC初心者向けのPC構成提案アプリ。

## コンセプト
- 予算と用途を選ぶだけ
- 失敗しにくいPC構成を提案
- 将来的にアフィリエイト導線と広告導線を組み込む

## 現在の実装
- 静的フロントエンドMVP
- 用途別・予算別のおすすめ構成表示
- 構成理由の説明表示
- 将来の購入導線 / 広告枠のプレースホルダー

## ファイル
- `site/index.html`
- `site/styles.css`
- `site/app.js`
